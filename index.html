AI Agent · EdTech

AI-Powered Exam Analyzer

A government education client needed to grade thousands of handwritten exam papers faster and more consistently. Teachers were spending hours on manual marking with no way to track student weaknesses or auto-generate targeted revision material.

Built a multi-agent pipeline using Llama 3.2 Vision for OCR on handwritten answers and NVIDIA Nemotron for reasoning-based rubric evaluation. Designed a RAG retrieval layer with ChromaDB and LangChain to power a question generator that uses K-means clustering (scikit-learn) to identify weak topics and produce personalised remedial papers automatically via a Streamlit dashboard.

RAG Chatbot · MarTech

Ad Intelligence Platform

A digital marketing team was spending hours manually pulling campaign reports from Meta Ads Manager. They had no quick way to check performance, compare week-over-week metrics, or get alerted when a campaign started underperforming.

Built a Flask-based chatbot powered by Llama 3.1 that lets marketers query ad data in plain English. APScheduler fetches from Meta's Graph API on a recurring schedule, ChromaDB stores and indexes the data, and NVIDIA Reranker sharpens retrieval accuracy. Integrated n8n for workflow orchestration and a Telegram bot via python-telegram-bot for real-time alerts when KPIs cross thresholds.

Market Intelligence Engine

A product team needed to track fast-moving consumer trends across social media, forums, and news — but manually monitoring dozens of sources produced unstructured noise that was impossible to act on in time.

AI Agent · EdTech

Chinese Handwriting Analyzer

Teachers grading Traditional Chinese handwritten compositions spend hours identifying character errors manually. Existing OCR tools struggle with shape-similar characters (e.g. 未/末, 己/已/巳) and provide inconsistent results across runs — making them unreliable for formal grading.

Built an end-to-end pipeline using PaddleOCR v5 with OpenCV preprocessing for layout detection and character recognition (93.29% accuracy). Fine-tuned a ResNet34 CNN with Center Loss on 79,760 synthetic images as a second-pass visual verifier for shape-similar characters (81.6% accuracy). Integrated GPT-4.1 API for context-aware spelling correction with batched validation, cutting API calls by 80% and RAM from ~553 MB to ~192 MB. Deterministic grading via temperature=0 ensures reproducible results. Generates PDF reports with teacher feedback.
